# Matrix für Verhaltens-Mapping

| Basis-Emotion | Bewegung: Niedrig ($0.0 - 0.3$) | Bewegung: Mittel ($0.3 - 0.7$) | Bewegung: Hoch ($0.7 - 1.0$) |
| :--- | :--- | :--- | :--- |
| **angry** | **Kaltes Fressen / Groll**<br>*(Unterdrückte, kontrollierte Wut; potenziell kalkuliert)* | **Bestimmtheit / Protest**<br>*(Konstruktiver Ärger, argumentierende Gestik)* | **Aggression / Wutausbruch**<br>*(Akuter Kontrollverlust, physische Drohhaltung)* |
| **disgust** | **Stille Verachtung**<br>*(Arroganz, innere Vorurteile, subtiles Abwenden)* | **Kritik / Ablehnung**<br>*(Sichtbares Unbehagen, wertende Kommunikation)* | **Physischer Ekel (Repulsion)**<br>*(Akuter Ekel- oder Brechreiz, panische Abwehrbewegung)* |
| **fear** | **Schockstarre / Paralyse**<br>*(Freeze-Reaktion, totale Überforderung/Schock)* | **Sorge / Skepsis**<br>*(Besorgtheit, vorsichtiges Abwägen)* | **Panik / Hysterie**<br>*(Fluchtreflex, Orientierungslosigkeit, akuter Stress)* |
| **happy** | **Zufriedenheit / Gelassenheit**<br>*(Innere Ruhe, stilles Genießen, Entspannung)* | **Freundlichkeit / Charisma**<br>*(Gute Konversation, einladende, offene Gestik)* | **Euphorie / Ekstase**<br>*(Überschwänglichkeit, Triumph, Feierstimmung)* |
| **neutral** | **Fokus / Konzentration**<br>*(Tiefe Aufmerksamkeit, Apathie oder mentaler Ruhemodus)* | **Alltagsmodus / Baseline**<br>*(Normale Aufmerksamkeit, entspanntes Zuhören)* | **Innere Unruhe / Maskierung**<br>*(Fidgeting: Gesicht bleibt ruhig, aber Hände verraten extreme Nervosität)* |
| **sad** | **Resignation / Depression**<br>*(In sich gekehrt, schwerer Energieverlust, tiefe Trauer)* | **Wehmut / Melancholie**<br>*(Sichtbares Bedauern, erzählende Trauer)* | **Verzweiflung / Agitierte Trauer**<br>*(Hilferuf, verzweifeltes Ringen nach Worten oder Halt)* |
| **surprise** | **Verblüffung / Staunen**<br>*(Sprachlosigkeit, intensive kognitive Verarbeitung)* | **Interesse / Neugier**<br>*(Unerwartete Erkenntnis, positives "Aha"-Erlebnis)* | **Erschrecken / Schrecksekunde**<br>*(Physischer Startle-Reflex, plötzliche wahrgenommene Bedrohung)* |
